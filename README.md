## 📖 Sobre o Projeto

O **MyContacts** é uma aplicação completa para gestão de contatos. O projeto foi desenvolvido de ponta a ponta durante o curso **JStack**, ministrado pelo Mateus Silva. 

O objetivo principal foi aprender a construir uma arquitetura sólida, sem depender de frameworks "mágicos" no início, entendendo conceitos profundos de Node.js, React e a comunicação entre ambos.

## 🛠 Tecnologias

O projeto utiliza o que há de mais moderno no ecossistema JavaScript:

[![Tecnologias](https://skillicons.dev/icons?i=js,react,styledcomponents,nodejs,postgres,docker)](https://skillicons.dev)

- **Front-end:** React.js, Styled Components.
- **Back-end:** Node.js.
- **Banco de Dados:** PostgreSQL & Docker.

## 📂 Estrutura de Pastas

- `/fe`: Código fonte da aplicação Front-end (React).
- `/api`: Código fonte da API REST (Node.js).
